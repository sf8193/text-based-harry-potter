# text-based-harry-potter
text based project

To run the game type you need to have leiningen installed. Type: lein run
  
  You can use commands like:
  look
  north
  south
  east
  west
  grab (item)
  drop (item)
  trade (item)
  fight (basilisk/voldemort)
  talk (Haggrid/Dumbledore)
  map
  lumos
  bag
  eat (item)
